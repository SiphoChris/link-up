routes/end-points



HTTP method Path Description
Get / (home) gets the home page that displays the routes

users

HTTP method Path Description
Get /api/users/all gets all the users
Get /api/users/:id gets a user by id
Post /api/users/register creates a new user
Post /api/users/login logs in a user
Patch /api/users/update/:id updates a user's data
Delete /api/users/delete/:id deletes a user by id

comments

HTTP method Path Description
Get /api/comments/all gets all the comments
Get /api/comments/from/:post_id gets a single comment
Post /api/comments/create creates a new comment
Patch /api/comments/update/:id updates comment data
Delete /api/comments/delete/:id deletes a comment by id

Post /api/posts/create creates a new posts
Patch /api/posts/update/:id updates posts data
