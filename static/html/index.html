routes/end-points


all my users routes have a base route of /api/users


all my comments routes have a base route of /api/comments


all my posts routes have a base route of /api/posts


This is a best practice that I have picked up


HTTP method Path Description
Get / (home) gets the home page that displays the routes

users

HTTP method Path Description
Get /all gets all the users
Get /:id gets a user by id
Post /register creates a new user
Post /login logs in a user
Patch /update/:id updates a user's data
Delete /delete/:id deletes a user by id

comments

HTTP method Path Description
Get /all gets all the comments
Get /:id gets a single comment
Post /create creates a new comment
Patch /update/:id updates comment data
Delete /delete/:id deletes a comment by id

posts

HTTP method Path Description
Get /all gets all the posts
Get /:id gets a single posts
Post /create creates a new posts
Patch /update/:id updates posts data
Delete /delete/:id deletes a posts by id
